Practicar conceptos básicos de programación mediante ejercicios académicos y ejemplos de código.

- Variables y tipos de datos
- Condicionales
- Ciclos
- Métodos
- Programación orientada a objetos
- Ejercicios básicos en Python

## Lenguajes utilizados
